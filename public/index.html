Parent workflow

Simple enough for any parent to use

  1. Pick a sport template. Download the front and back SVG files.
  2. Open in Canva or another editor. Upload the school/team logo and player photo.
  3. Edit text. Replace name, number, grade, position, stats, fun facts, and parent message.
  4. Export for printing. Save as PDF print or PNG at 300 DPI.
  5. Print or order cards. Use cardstock, a local print shop, or a trading card printer.

Questions parents will ask

FAQ

Can parents customize these themselves?

Yes. The templates are SVG files, so parents can import them into Canva, Photopea, Figma, Illustrator, or Inkscape and replace the photo, logo, and text.

Are these print ready?

The card design is built at 825 × 1125 pixels, which matches 2.75 × 3.75 inches at 300 DPI including bleed. Keep important text inside the border before printing.

Can schools use their own logo?

Yes. Each template has a team logo area on the front and back. Parents should only use logos they have permission to use.

Can this become a real store?

Yes. Replace the demo buttons with Shopify, Stripe payment links, Etsy digital downloads, Gumroad, or a custom checkout.
